Customer Dispute Classification

Customer Complaints

Deep Learning
Tensorflow Django RESTful API

This project leverages Natural Language Processing (NLP) to classify customer complaints submitted to banks and predict whether the customer is likely to dispute a product or service. The application also summarizes the complaints for quick understanding. It combines Machine Learning with a RESTful API built in Django to create an end-to-end system that can be integrated into customer support tools for faster resolution and product improvement insights.

Project Details:

  • Takes raw customer complaints as input and predicts whether a dispute is likely to occur using trained Machine Learning and Deep Learning models (including Bi-LSTM).
  • Integrates Hugging Face’s transformers summarization pipeline to generate a concise overview of the complaint, saving time for analysts and customer support staff.

Credit Risk Prediction

Predicts Credit Risk of borrowers

Machine Learning
Python Sci-Kit Learn Pandas XGBoost

Developed a machine learning model that predicts the potential credit risk of borrowers based on their information. The model achieved 87% accuracy in classifying if a potential borrower will default on the loan or not, which can give insights on which kind of clients to target for loans to reduce the amount of defaults.

Project Details:

  • Applied undersampling to balance a 30,000+ row dataset.
  • Implemented XGB model for binary classification
  • Applied GridSearchCV for hyperparameter tuning.

Key Achievements:

  • Got up to a 87% overall model accuracy.
  • Gained insights on the features that are important to determine loan defaults.
